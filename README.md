
# Currency Exchange Web Application

A web application that allows users to convert currencies in real-time using the Exchange Rate API. The app provides a user-friendly interface to select currencies, enter an amount, and view the conversion result.

## Features
- Real-time currency conversion.
- Supports a wide range of currencies.
- Easy-to-use dropdown menus to select currencies.
- Displays the converted amount instantly.

## Technologies Used
- **HTML**: For creating the structure of the web page.
- **CSS**: For styling the application.
- **JavaScript**: For handling currency conversion logic and interacting with the API.
- **Exchange Rate API**: To get real-time currency exchange rates.
